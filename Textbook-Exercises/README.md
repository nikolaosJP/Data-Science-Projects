# Book Exercises

This repository contains solutions to exercises from various statistics and machine learning textbooks.

## Books Included

### Applied Predictive Modeling
Solutions to exercises from "Applied Predictive Modeling" by Max Kuhn and Kjell Johnson.

### Introduction to Statistical Learning (2nd Edition)
Solutions to exercises from "An Introduction to Statistical Learning with Applications in R" (2nd Edition) by Gareth James, Daniela Witten, Trevor Hastie, and Robert Tibshirani.

## Purpose

This repository serves as a collection of worked solutions and code implementations for exercises from these foundational texts in statistical learning and predictive modeling. The solutions are organized by book and chapter for easy reference.
