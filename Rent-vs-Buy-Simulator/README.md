# Rent vs Buy Simulator

An interactive financial calculator that helps you make informed decisions about renting versus buying a home. Built with React, Vite, and Recharts to visualize long-term financial outcomes under different scenarios.

## What It Does

Compares the financial implications of renting versus buying a property over time, considering:
- Mortgage payments (principal and interest)
- Property taxes and maintenance costs
- Tax deductions for homeownership
- Opportunity cost of invested capital
- Different house types (Long-term Quality, ZEH, Energy Efficient, Standard)
- Japanese tax bracket calculations

## Features

- Interactive parameter adjustments (purchase price, down payment, interest rates, etc.)
- Multi-year projections with visual charts
- Net worth comparison between renting and buying scenarios
- Breakdown of equity buildup, interest payments, and tax benefits
- Responsive design with Tailwind CSS

## Tech Stack

- React 18
- Vite
- Recharts for data visualization
- Tailwind CSS for styling
- Lucide React for icons

## Getting Started

```bash
./run.sh
```

The script automatically installs dependencies if needed and starts the development server. Visit `http://localhost:5173` to use the simulator.

## Use Case

Ideal for anyone considering homeownership, particularly useful for understanding Japanese housing market dynamics and tax implications.
